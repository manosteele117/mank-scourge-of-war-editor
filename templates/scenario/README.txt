======================
For local use/testing:
=======================
===Use as a Mod===
This folder can be be placed into <Game Dir>/Base/Mods/ to allow selection in the modifications screen. It contains everything you need to run your scenario.

===Use as a Scenario===
If this folder generated with only a 'Scenarios' folder, you can place the contents of that within <Game Dir>/Base/Scenarios/ directly. This means you don't have to activate a new mod, and means your existing OOB and Language files cover everything in this scenario. If this generated with a Layout/ or OOBs/ folder, this will not work.

===With other mods===
If you made your scenario/oob with mod files (new drills, or new master OOB) then you must load that mod.
